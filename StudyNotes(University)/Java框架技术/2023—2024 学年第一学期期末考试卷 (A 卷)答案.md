### 四、名词解释
**(每小题 2 分，本大题共 10 分)**

1. **Ioc—Inversion of Control**，即“控制反转”，不是什么技术，而是一种设计思想。（1 分）
   控制反转是说创建对象的控制权进行转移，以前创建对象的主动权和创建时机是由自己把控的，而现在这种权力转移到第三方。（1 分）

2. **AOP 为 Aspect Oriented Programming 的缩写**，意为：面向切面编程，通过预编译方式和运行期间动态代理实现程序功能的统一维护的一种技术。（1 分）
   利用 AOP 可以对业务逻辑的各个部分进行隔离，从而使得业务逻辑各部分之间的耦合度降低，提高程序的可重用性，同时提高了开发的效率。（1 分）

3. **在 Spring 框架中，Bean 的生命周期主要包括四个部分**：
   实例化（Instantiation）（0.5 分）
   属性赋值（Populate）（0.5 分）
   初始化（Initialization）（0.5 分）
   销毁（Destruction）（0.5 分）

4. **MVC（模型—视图—控制）是一种框架模式**。（1 分）
   使用 MVC 的目的是将 M 和 V 的实现代码分离，从而使同一个程序可以使用不同的表现形式。（1 分）

5. **MyBatis 的核心组件主要有四个**：
   (1) SqlSessionFactoryBuilder（构造器）：它会根据配置或者代码来生成 SqlSessionFactory，采用的是分步构建的 Builder 模式。（0.5 分）
   (2) SqlSessionFactory（工厂接口）：依靠它来生成 SqlSession，使用的是工厂模式。（0.5 分）
   (3) SqlSession（会话）：一个既可以发送 SQL 执行返回结果，也可以获取 Mapper 的接口。在现有的技术中，一般我们会让其在业务逻辑代码中“消失”，而使用的是 MyBatis 提供的 SQL Mapper 接口编程技术，它能提高代码的可读性和可维护性。（0.5 分）
   (4) SQL Mapper（映射器）：MyBatis 新设计存在的组件，它由一个 Java 接口和 XML 文件（或注解）构成，需要给出对应的 SQL 和映射规则。它负责发送 SQL 去执行，并返回结果。（0.5 分）

### 五、简答题
**(每小题 5 分，本大题共 20 分，回答合理即可得分)**

1. **主要七大模块**：
   2. Spring AOP 面向切面编程
   3. Spring ORM Hibernate|mybatis|JDO
   4. Spring Core 提供 bean 工厂 IOC
   5. Spring Dao JDBC 支持
   6. Spring Context 提供了关于 UI 支持，邮件支持等
   7. Spring Web 提供了 web 的一些工具类的支持
   8. Spring MVC 提供了 web mvc, webviews, jsp, pdf, export
   (写出五个即可)

9. @Controller、@RequestMapping、@Resource、@Autowired、@ResponseBody、@Repository、@Component、@RequestParam、@PathVariable、@ModelAttribute、@SessionAttributes 等等
   (写出 5 个即可)

10. **@RequestBody**
   11) 该注解用于读取 Request 请求的 body 部分数据，使用系统默认配置的 HttpMessageConverter 进行解析，然后把相应的数据绑定到要返回的对象上；
   12) 再把 HttpMessageConverter 返回的对象数据绑定到 controller 中方法的参数上。
   **@ResponseBody**
   该注解用于将 Controller 的方法返回的对象，通过适当的 HttpMessageConverter 转换为指定格式后，写入到 Response 对象的 body 数据区。

13. **@Autowired** 可用于：构造函数、成员变量、Setter 方法
   @Autowired 默认是按照类型装配注入的，默认情况下它要求依赖对象必须存在（可以设置它 required 属性为 false）。
   **@Resource** 默认是按照名称来装配注入的，只有当找不到与名称匹配的 bean 才会按照类型来装配注入。

   ```xml
   <filter>
       <filter-name>filter</filter-name>
       <filter-class>org.springframework.web.filter.CharacterEncodingFilter</filter-class>
       <init-param>
           <param-name>encoding</param-name>
           <param-value>utf-8</param-value>
       </init-param>
   </filter>
   <filter-mapping>
       <filter-name>filter</filter-name>
       <url-pattern>/*</url-pattern>
   </filter-mapping>
   ```

       <servlet-name>springmvc</servlet-name>
       <servlet-class>org.springframework.web.servlet.DispatcherServlet</servlet-class>

           <param-value>classpath*:springmvc_servlet.xml</param-value>
       </init-param>

       <servlet-name>springmvc</servlet-name>

   </servlet-mapping>
