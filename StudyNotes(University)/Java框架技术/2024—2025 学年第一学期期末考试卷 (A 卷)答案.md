### 四、名词解释
**(每小题 2 分，本大题共 10 分)**

1. **Ioc—Inversion of Control**，即“控制反转”，不是什么技术，而是一种设计思想。（1 分）
   控制反转是说创建对象的控制权进行转移，以前创建对象的主动权和创建时机是由自己把控的，而现在这种权力转移到第三方。（1 分）

2. **AOP 为 Aspect Oriented Programming 的缩写**，意为：面向切面编程，通过预编译方式和运行期间动态代理实现程序功能的统一维护的一种技术。（1 分）
   利用 AOP 可以对业务逻辑的各个部分进行隔离，从而使得业务逻辑各部分之间的耦合度降低，提高程序的可重用性，同时提高了开发的效率。（1 分）

3. **JDBCTenplate 就是 Spring 对 JDBC 内置的封装**，通俗点说就是 Spring 对 JDBC 的封装的模板。（1 分）
   使用 spring 的注入功能，可以把 Datasource 注册到 JdbcTemplate 之中，相比较于传统的 JDBC，大大简化了数据库的操作。（1 分）

4. **MVC（模型—视图—控制）是一种框架模式**。（1 分）
   使用 MVC 的目的是将 M 和 V 的实现代码分离，从而使同一个程序可以使用不同的表现形式。（1 分）

5. **Ajax 即 “Asynchronous Javascript And XML”（异步 JavaScript 和 XML）**，是指一种创建交互式、快速动态网页应用的网页开发技术，无需重新加载整个网页的情况下，能够更新部分网页的技术。（1 分）
   通过在后台与服务器进行少量数据交换，Ajax 可以使网页实现异步更新。（1 分）

**(每小题 5 分，本大题共 20 分)**

1. **主要七大模块**：

   3. Spring ORM Hibernate|mybatis|JDO
   4. Spring Core 提供 bean 工厂 IOC
   5. Spring Dao JDBC 支持
   6. Spring Context 提供了关于 UI 支持，邮件支持等
   7. Spring Web 提供了 web 的一些工具类的支持
   8. Spring MVC 提供了 web mvc, webviews, jsp, pdf, export
   (写出五个即可，每个 1 分)

9. @Controller、@RequestMapping、@Resource、@Autowired、@ResponseBody、@Repository、@Component、@RequestParam、@PathVariable、@ModelAttribute、@SessionAttributes 等等
   (写出 5 个即可，每个 1 分)

10. **@RequestBody**
   11) 该注解用于读取 Request 请求的 body 部分数据，使用系统默认配置的 HttpMessageConverter 进行解析，然后把相应的数据绑定到要返回的对象上；(2 分)
   12) 再把 HttpMessageConverter 返回的对象数据绑定到 controller 中方法的参数上。(2 分)
   **@ResponseBody**
   该注解用于将 Controller 的方法返回的对象，通过适当的 HttpMessageConverter 转换为指定格式后，写入到 Response 对象的 body 数据区。(1 分)

13. **@Autowired** 可用于：构造函数、成员变量、Setter 方法 (1 分)
   @Autowired 默认是按照类型装配注入的，默认情况下它要求依赖对象必须存在（可以设置它 required 属性为 false）。(2 分)
   **@Resource** 默认是按照名称来装配注入的，只有当找不到与名称匹配的 bean 才会按照类型来装配注入。(2 分)

       <filter-name>filter</filter-name>
