# 《概率论与数理统计》详细复习指南 (含例题)

## 第一章：概率论的基本概念

这一章是基础，重点在于**事件的关系运算**和**三大公式**（加法、乘法、全概率/贝叶斯）。

### 1. 随机试验、样本空间与随机事件

*   **随机试验 (E)**：
    1.  可在相同条件下重复进行。
    2.  每次试验的可能结果不止一个，且能事先明确所有可能结果。
    3.  进行一次试验之前不能确定哪一个结果会出现。
    *   **例**：抛掷一枚均匀硬币。
*   **样本空间 ($S$)**：随机试验所有可能结果的集合。通常用大写字母$S$表示。
    *   **例**：抛掷一枚均匀硬币，样本空间 $S = \{ \text{正面}, \text{反面} \}$。
*   **随机事件**：样本空间的子集。通常用大写字母$A, B, C$等表示。
    *   **例**：抛掷一枚均匀硬币，“出现正面”是一个事件。
*   **事件的关系与运算**：
    *   **包含** $A \subset B$：若$A$发生必导致$B$发生。
    *   **相等** $A = B$：若$A \subset B$ 且 $B \subset A$。
    *   **和事件（并集）** $A \cup B$：$A$和$B$至少一个发生。
    *   **积事件（交集）** $A \cap B$（或简写为$AB$）：$A$和$B$同时发生。
    *   **差事件** $A - B$：$A$发生且$B$不发生。
    *   **互斥（互不相容）** $AB = \emptyset$：$A$与$B$不能同时发生。
    *   **对立（逆事件）** $\bar{A}$：$A$不发生。注意：$A \cup \bar{A} = S, A \cap \bar{A} = \emptyset$。
    *   **运算律**：
        *   交换律：$A \cup B = B \cup A$, $AB = BA$。
        *   结合律：$(A \cup B) \cup C = A \cup (B \cup C)$, $(AB)C = A(BC)$。
        *   分配律：$A \cup (BC) = (A \cup B)(A \cup C)$, $A(B \cup C) = AB \cup AC$。
        *   **德·摩根律 (De Morgan)**：非常重要，常用于化简。
            $$ \overline{A \cup B} = \bar{A} \cap \bar{B} $$
            $$ \overline{A \cap B} = \bar{A} \cup \bar{B} $$

    *   **例题1.1**：掷一枚骰子，观察出现的点数。
        设事件 $A = \text{“点数不大于4”}$，事件 $B = \text{“点数为偶数”}$。
        则 $S = \{1, 2, 3, 4, 5, 6\}$。
        $A = \{1, 2, 3, 4\}$。
        $B = \{2, 4, 6\}$。
        *   $A \cup B = \{1, 2, 3, 4, 6\}$（点数不大于4或为偶数）。
        *   $AB = \{2, 4\}$（点数不大于4且为偶数）。
        *   $A - B = \{1, 3\}$（点数不大于4但不是偶数）。
        *   $\bar{A} = \{5, 6\}$（点数大于4）。
        *   验证德·摩根律：$\overline{A \cup B} = \{5\} = \bar{A} \cap \bar{B}$ (因为 $\bar{B} = \{1, 3, 5\}$)。

### 2. 概率的定义

*   **古典概型**：
    *   特点：
        1.  样本空间只包含有限个元素。
        2.  试验中每个基本事件（每种可能的结果）发生的可能性相同（等可能性）。
    *   公式：
        $$ P(A) = \frac{\text{事件}A\text{包含的样本点数}}{\text{样本空间}S\text{的总样本点数}} $$
    *   **例题1.2**：从一批产品中任选三件，记录出现正品（N）与次品（D）的情况。
        $S = \{NNN, NND, NDN, DNN, NDD, DDN, DND, DDD\}$。
        假设每种情况等可能。
        设事件 $A = \text{“恰好有两件正品”}$。
        $A = \{NND, NDN, DNN\}$。
        $P(A) = \frac{3}{8}$。
*   **几何概型**：样本点无限，但概率只与度量（长度、面积、体积）成比例。
    *   公式：$P(A) = \frac{\text{构成事件}A\text{区域的度量}}{\text{样本空间}S\text{区域的度量}}$。
    *   **例题1.3**：两人相约在0到$T$时间内会面，先到者等候另一个人，经过时间$t$ ($t<T$) 后离去。问两人能会面的概率。
        设两人到达时间分别为$x, y$。样本空间为$S = \{(x,y) | 0 \le x \le T, 0 \le y \le T\}$，这是一个面积为$T^2$的正方形。
        能会面的条件是 $|x-y| \le t$，即 $-t \le x-y \le t$。
        这是一个在正方形内的带状区域。计算该区域面积：$T^2 - (T-t)^2$。
        $P(\text{会面}) = \frac{T^2 - (T-t)^2}{T^2} = 1 - (1-\frac{t}{T})^2$。
*   **公理化性质**：
    *   $0 \le P(A) \le 1$。
    *   $P(S) = 1$，$P(\emptyset) = 0$。
    *   若$A_1, A_2, ...$是互不相容的事件，则 $P(\cup_{i=1}^\infty A_i) = \sum_{i=1}^\infty P(A_i)$。
    *   **加法公式**：$P(A \cup B) = P(A) + P(B) - P(AB)$。
        *   若$A,B$互斥，则 $P(A \cup B) = P(A) + P(B)$。
    *   **减法公式**：$P(A - B) = P(A) - P(AB)$。
    *   **例题1.4**：已知 $P(A)=0.5, P(B)=0.6, P(A \cup B)=0.9$。
        求 $P(AB)$。
        根据加法公式：$P(A \cup B) = P(A) + P(B) - P(AB)$。
        $0.9 = 0.5 + 0.6 - P(AB)$。
        $P(AB) = 1.1 - 0.9 = 0.2$。

### 3. 条件概率与独立性 (**核心**)

*   **条件概率**：在$A$发生的条件下$B$发生的概率。
    $$ P(B|A) = \frac{P(AB)}{P(A)} \quad (P(A)>0) $$
    *   **例题1.5**：一盒子装有4只产品，其中3只一等品，1只二等品。从中取产品两次，每次任取一只，作不放回抽样。设事件$A = \text{“第一次取到一等品”}$，事件$B = \text{“第二次取到一等品”}$。求 $P(B|A)$。
        $S$中共有 $4 \times 3 = 12$ 种取法。
        $A$包含的样本点数：第一次取到一等品，则有3种选择，第二次再取3种，共$3 \times 3 = 9$种。 $P(A) = \frac{9}{12} = \frac{3}{4}$。
        $AB$包含的样本点数：第一次取到一等品，有3种选择，第二次取到一等品，则只剩下2种一等品。共 $3 \times 2 = 6$种。 $P(AB) = \frac{6}{12} = \frac{1}{2}$。
        $$P(B|A) = \frac{P(AB)}{P(A)} = \frac{1/2}{3/4} = \frac{2}{3}$$
*   **乘法定理**：
    $$ P(AB) = P(A)P(B|A) $$
    *   推广：$P(A_1A_2...A_n) = P(A_1)P(A_2|A_1)P(A_3|A_1A_2)...P(A_n|A_1...A_{n-1})$。
*   **独立性**：
    *   定义：$A, B$ 独立 $\iff P(AB) = P(A)P(B)$。
    *   **区分**：独立与互斥是不同的概念。互斥是“不能同时发生”，独立是“发生与否互不影响”。
    *   性质：若$A, B$独立，则 $A$与$\bar{B}$、$\bar{A}$与$B$、$\bar{A}$与$\bar{B}$ 也都独立。
    *   **例题1.6**：投掷一枚均匀色子，事件 $A = \text{“点数不大于4”}$，事件 $B = \text{“点数为奇数”}$。
        $P(A) = \frac{4}{6} = \frac{2}{3}$。
        $P(B) = \frac{3}{6} = \frac{1}{2}$。
        $AB = \{1, 3\}$，$P(AB) = \frac{2}{6} = \frac{1}{3}$。
        因为 $P(A)P(B) = \frac{2}{3} \times \frac{1}{2} = \frac{1}{3} = P(AB)$，所以$A, B$独立。
*   **全概率公式**（由因推果）：
    设 $B_1, ..., B_n$ 是样本空间的一个划分（即 $B_i$ 互斥，且 $\cup_{i=1}^n B_i = S$），则对任意事件 $A$：
    $$P(A) = \sum_{i=1}^n P(B_i)P(A|B_i)$$
    *   **例题1.7**：一批产品由三个工厂生产，各自占30%，50%，20%。已知这三个工厂的产品次品率分别为2%，1%，1%。问从这批产品中任取一件是次品的概率是多少？
        设 $A = \text{“取到次品”}$。
        设 $B_1 = \text{“产品来自第一工厂”}$，$B_2 = \text{“产品来自第二工厂”}$，$B_3 = \text{“产品来自第三工厂”}$。
        已知 $P(B_1)=0.3, P(B_2)=0.5, P(B_3)=0.2$。
        已知 $P(A|B_1)=0.02, P(A|B_2)=0.01, P(A|B_3)=0.01$。
        根据全概率公式：
        $P(A) = P(B_1)P(A|B_1) + P(B_2)P(A|B_2) + P(B_3)P(A|B_3)$
        $P(A) = 0.3 \times 0.02 + 0.5 \times 0.01 + 0.2 \times 0.01 = 0.006 + 0.005 + 0.002 = 0.013$。
*   **贝叶斯公式 (Bayes' Theorem)**（执果索因，已知结果推测原因的概率）：
    $$ P(B_i|A) = \frac{P(B_i)P(A|B_i)}{\sum_{j=1}^n P(B_j)P(A|B_j)} $$
    *   **例题1.8**：接上例，若已知取到的产品是次品，问它来自第一工厂的概率是多少？
        已知 $P(A)=0.013$（由上例计算）。
        求 $P(B_1|A)$。
        根据贝叶斯公式：
        $$ P(B_1|A) = \frac{P(B_1)P(A|B_1)}{P(A)} = \frac{0.3 \times 0.02}{0.013} = \frac{0.006}{0.013} \approx 0.4615 $$

## 第二章：随机变量及其分布

这一章重点是**常见分布的定义、分布律/概率密度函数、图像特点**，以及**分布函数的性质**。

### 1. 随机变量 (R.V.)

*   **定义**：一个变量，其取值随着试验结果的变化而变化，且取值具有随机性。它是一个函数 $X(e)$，将样本空间的样本点 $e$ 映射到实数轴上的值。
*   **类型**：
    *   **离散型随机变量**：取值是有限个或可列无限个。
    *   **连续型随机变量**：取值充满某个区间，通常有无数个值。

### 2. 离散型随机变量

*   **描述方式**：**分布律**（或概率质量函数 PMF） $P\{X=x_k\} = p_k$。
    *   性质：$p_k \ge 0$，$\sum_k p_k = 1$。
    *   **例题2.1**：掷两颗骰子，记其点数之和为$X$。求$X$的分布律。
        $X$的可能取值为$2, 3, ..., 12$。
        $P\{X=2\} = P\{(1,1)\} = \frac{1}{36}$。
        $P\{X=3\} = P\{(1,2), (2,1)\} = \frac{2}{36}$。
        ...
        $P\{X=7\} = P\{(1,6), (2,5), (3,4), (4,3), (5,2), (6,1)\} = \frac{6}{36}$。
        ...
        $P\{X=12\} = P\{(6,6)\} = \frac{1}{36}$。
*   **常见分布**（**必须熟记**）：
    1.  **0-1分布 (Bernoulli)**：$X \sim B(1, p)$。$P\{X=1\}=p, P\{X=0\}=1-p$。
    2.  **二项分布 $X \sim B(n, p)$**：$n$重伯努利试验中成功的次数。
        $$ P\{X=k\} = C_n^k p^k (1-p)^{n-k}, \quad k=0,1,...,n $$
        *   **例题2.2**：某产品合格率为0.8。从生产线上抽取5件产品，记合格品件数为$X$。求$X$的分布律。
            $X \sim B(5, 0.8)$。
            $P\{X=k\} = C_5^k (0.8)^k (0.2)^{5-k}, \quad k=0,1,2,3,4,5$。
    3.  **泊松分布 $X \sim P(\lambda)$**：单位时间内或单位面积内稀有事件发生的次数。
        $$ P\{X=k\} = \frac{\lambda^k e^{-\lambda}}{k!}, \quad k=0,1,2,... $$
        *   **例题2.3**：某呼叫中心每分钟平均接到2个电话。假设电话呼入服从泊松分布，求一分钟内接到3个电话的概率。
            $X \sim P(2)$。
            $P\{X=3\} = \frac{2^3 e^{-2}}{3!} = \frac{8e^{-2}}{6} = \frac{4e^{-2}}{3}$。
        *   *泊松定理*：当$n$很大（通常$n>20$），$p$很小（通常$p<0.05$）时，二项分布$B(n, p)$ 可用泊松分布$P(np)$近似。

### 3. 连续型随机变量

*   **描述方式**：**概率密度函数 (PDF) $f(x)$**。
    *   性质：
        1.  $f(x) \ge 0$。
        2.  $\int_{-\infty}^{+\infty} f(x)dx = 1$。
    *   概率计算：$P\{a < X < b\} = \int_{a}^{b} f(x)dx$。
        *   **注意**：对于连续型随机变量，$P\{X=a\}=0$。所以 $P\{a < X < b\} = P\{a \le X < b\} = P\{a < X \le b\} = P\{a \le X \le b\}$。
*   **分布函数 (CDF) $F(x)$**：
    *   定义：$F(x) = P\{X \le x\} = \int_{-\infty}^{x} f(t)dt$。
    *   性质：
        1.  $0 \le F(x) \le 1$。
        2.  $F(x)$ 是单调不减函数。
        3.  $F(x)$ 是右连续的。
        4.  $\lim_{x \to -\infty} F(x) = 0, \lim_{x \to +\infty} F(x) = 1$。
    *   关系：在$f(x)$的连续点处，$F'(x) = f(x)$。
*   **常见分布**（**必须熟记**）：
    1.  **均匀分布 $X \sim U(a, b)$**：
        $$ f(x) = \begin{cases} \frac{1}{b-a}, & a<x<b \\ 0, & \text{其他} \end{cases} $$
        $$ F(x) = \begin{cases} 0, & x \le a \\ \frac{x-a}{b-a}, & a<x<b \\ 1, & x \ge b \end{cases} $$
        *   **例题2.4**：某公交车每隔10分钟一班，乘客到达车站的时间服从$U(0, 10)$。问乘客等待时间超过7分钟的概率。
            设等待时间为$X \sim U(0, 10)$。
            $f(x) = \frac{1}{10}$，当$0<x<10$。
            $P\{X>7\} = \int_7^{10} \frac{1}{10} dx = [\frac{x}{10}]_7^{10} = \frac{10}{10} - \frac{7}{10} = \frac{3}{10} = 0.3$。
    2.  **指数分布 $X \sim E(\lambda)$**：常用于描述无记忆性事件（如电子元件寿命、电话等待时间）。
        $$ f(x) = \begin{cases} \lambda e^{-\lambda x}, & x>0 \\ 0, & x \le 0 \end{cases} $$
        $$ F(x) = \begin{cases} 0, & x \le 0 \\ 1-e^{-\lambda x}, & x>0 \end{cases} $$
        *   性质：具有**无记忆性** $P\{X>s+t|X>s\} = P\{X>t\}$。
        *   **例题2.5**：某种电子元件的寿命服从参数为$\lambda=0.001$的指数分布（单位：小时）。
            求元件能正常工作1000小时以上的概率。
            $P\{X>1000\} = 1 - F(1000) = 1 - (1 - e^{-0.001 \times 1000}) = e^{-1} \approx 0.3679$。
            若已知元件已经工作了1000小时，问还能工作1000小时以上的概率。
            $P\{X>2000|X>1000\} = P\{X>1000\} = e^{-1}$ （利用无记忆性）。
    3.  **正态分布 $X \sim N(\mu, \sigma^2)$**：
        $$ f(x) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}} $$
        $$ F(x) = \Phi(\frac{x-\mu}{\sigma}) $$
        *   参数：$\mu$ 为均值，$\sigma^2$ 为方差。
        *   性质：关于 $x=\mu$ 对称。
        *   **标准化**：若 $X \sim N(\mu, \sigma^2)$，则 $Z = \frac{X-\mu}{\sigma} \sim N(0, 1)$。
        *   标准正态分布的CDF用 $\Phi(x)$ 表示，$P\{X \le x\} = \Phi(x)$。
        *   **计算**：
            1.  查标准正态分布表 ($\Phi(x)$)。
            2.  利用对称性：$\Phi(-x) = 1 - \Phi(x)$。
            3.  $P\{a < X < b\} = \Phi(\frac{b-\mu}{\sigma}) - \Phi(\frac{a-\mu}{\sigma})$。
        *   **例题2.6**：设某机床加工零件的长度$X \sim N(20, 0.5^2)$mm。问零件长度在19.5mm到20.5mm之间的概率。
            $\mu=20, \sigma=0.5$。
            $P\{19.5 < X < 20.5\} = P\{\frac{19.5-20}{0.5} < \frac{X-20}{0.5} < \frac{20.5-20}{0.5}\}$
            $= P\{-1 < Z < 1\} = \Phi(1) - \Phi(-1)$
            $= \Phi(1) - (1-\Phi(1)) = 2\Phi(1) - 1$
            $= 2 \times 0.8413 - 1 = 1.6826 - 1 = 0.6826$。

### 4. 随机变量函数的分布

*   **离散型随机变量函数**：
    *   方法：列出$X$的分布律，根据$Y=g(X)$算出$Y$的所有可能取值，然后将对应$X$值的概率合并到$Y$的相同取值上。
    *   **例题2.7**：设离散型随机变量$X$的分布律为：
    
        | $X$ | -1 | 0 | 1 | 2 |
        | :--- | :--- | :--- | :--- | :--- |
        | $P$ | 0.2 | 0.3 | 0.4 | 0.1 |
        求 $Y = X^2$ 的分布律。
        $Y$的可能取值：$(-1)^2=1, 0^2=0, 1^2=1, 2^2=4$。
        所以$Y$的取值为$0, 1, 4$。
        $P\{Y=0\} = P\{X=0\} = 0.3$。
        $P\{Y=1\} = P\{X=-1\} + P\{X=1\} = 0.2 + 0.4 = 0.6$。
        $P\{Y=4\} = P\{X=2\} = 0.1$。
        $Y$的分布律为：
        
        | $Y$ | 0 | 1 | 4 |
        | :-- | :- | :- | :- |
        | $P$ | 0.3 | 0.6 | 0.1 |
*   **连续型随机变量函数**：
    *   **分布函数法**（通用且重要）：
        1.  写出 $Y=g(X)$ 的分布函数 $F_Y(y) = P\{Y \le y\} = P\{g(X) \le y\}$。
        2.  将 $P\{g(X) \le y\}$ 转化为关于 $X$ 的概率，通过对 $X$ 的概率密度函数积分求得 $F_Y(y)$。
        3.  对 $F_Y(y)$ 求导即可得到 $Y$ 的概率密度函数 $f_Y(y) = F_Y'(y)$。
    *   **例题2.8**：设 $X \sim U(0, 1)$，求 $Y = e^X$ 的概率密度函数。
        $X$ 的概率密度函数 $f_X(x) = 1$ (当 $0<x<1$ 时，否则为0)。
        1.  求 $F_Y(y)$：$F_Y(y) = P\{Y \le y\} = P\{e^X \le y\}$。
            *   若 $y \le e^0=1$，则 $P\{e^X \le y\} = 0$。
            *   若 $y > e^1=e$，则 $P\{e^X \le y\} = 1$。
            *   若 $1 < y \le e$，则 $P\{e^X \le y\} = P\{X \le \ln y\} = \int_0^{\ln y} f_X(x)dx = \int_0^{\ln y} 1 dx = \ln y$。
            所以 $F_Y(y) = \begin{cases} 0, & y \le 1 \\ \ln y, & 1<y \le e \\ 1, & y > e \end{cases}$。
        2.  求 $f_Y(y)$：对 $F_Y(y)$ 求导。
            $$ f_Y(y) = F_Y'(y) = \begin{cases} \frac{1}{y}, & 1<y \le e \\ 0, & \text{其他} \end{cases} $$

## 第三章：多维随机变量

重点是**二维联合分布**、**边缘分布**、**独立性**的判断，以及**二维随机变量函数的期望**。

### 1. 联合分布

*   **二维离散型随机变量 $(X, Y)$**：
    *   **联合分布律** $p_{ij} = P\{X=x_i, Y=y_j\}$。
    *   性质：$p_{ij} \ge 0$, $\sum_i \sum_j p_{ij} = 1$。
    *   **例题3.1**：已知 $(X,Y)$ 的联合分布律为：
    
        | $X \setminus Y$ | 1 | 2 |
        | :-------------- | :- | :- |
        | 0 | 0.1 | 0.2 |
        | 1 | 0.3 | 0.4 |
*   **二维连续型随机变量 $(X, Y)$**：
    *   **联合概率密度函数** $f(x, y)$。
    *   性质：
        1.  $f(x, y) \ge 0$。
        2.  $\int_{-\infty}^{+\infty}\int_{-\infty}^{+\infty} f(x, y)dxdy = 1$。
    *   概率计算：$P\{(X,Y) \in G\} = \iint_G f(x,y)dxdy$（计算$f(x,y)$在区域$G$上的体积）。

### 2. 边缘分布

*   从联合分布求单个变量的分布。
*   **离散**：边缘概率等于所在行（或列）的联合概率之和。
    *   $P\{X=x_i\} = p_{i\cdot} = \sum_j p_{ij}$。
    *   $P\{Y=y_j\} = p_{\cdot j} = \sum_i p_{ij}$。
    *   **例题3.2**：接例题3.1。
        $P\{X=0\} = p_{01} + p_{02} = 0.1 + 0.2 = 0.3$。
        $P\{X=1\} = p_{11} + p_{12} = 0.3 + 0.4 = 0.7$。
        $P\{Y=1\} = p_{01} + p_{11} = 0.1 + 0.3 = 0.4$。
        $P\{Y=2\} = p_{02} + p_{12} = 0.2 + 0.4 = 0.6$。
*   **连续**：积分掉另一个变量。
    $$ f_X(x) = \int_{-\infty}^{+\infty} f(x, y)dy $$
    $$ f_Y(y) = \int_{-\infty}^{+\infty} f(x, y)dx $$
    *   **例题3.3**：设 $f(x,y) = 2$ ($0<x<y<1$)，否则为0。求 $f_X(x)$。
        $f_X(x) = \int_{-\infty}^{+\infty} f(x,y)dy = \int_x^1 2 dy = 2[y]_x^1 = 2(1-x)$, 当 $0<x<1$。
        否则 $f_X(x)=0$。

### 3. 相互独立性

*   **定义**：联合分布等于边缘分布的乘积。
    *   离散：$p_{ij} = p_{i\cdot} \times p_{\cdot j}$ 对所有$i,j$成立。
    *   连续：$f(x, y) = f_X(x) f_Y(y)$ 对所有$x,y$成立。
*   **判定技巧**：
    *   如果 $f(x,y)$ 可以分离变量写成 $g(x)h(y)$ 且定义域是矩形区域（如 $a<x<b, c<y<d$），则 $X, Y$ 独立。
    *   **例题3.4**：接例题3.1和3.2。
        $P\{X=0\} = 0.3, P\{Y=1\} = 0.4$。
        $p_{01} = 0.1$。
        $P\{X=0\}P\{Y=1\} = 0.3 \times 0.4 = 0.12 \ne 0.1$。
        所以 $X, Y$ 不独立。
    *   **例题3.5**：设 $f(x,y) = e^{-(x+y)}$ ($x>0, y>0$)，否则为0。判断 $X, Y$ 是否独立。
        $f_X(x) = \int_0^\infty e^{-(x+y)}dy = e^{-x} \int_0^\infty e^{-y}dy = e^{-x} [-e^{-y}]_0^\infty = e^{-x}$ ($x>0$)。
        同理 $f_Y(y) = e^{-y}$ ($y>0$)。
        因为 $f(x,y) = e^{-x}e^{-y} = f_X(x)f_Y(y)$，且定义域是矩形区域（$x>0, y>0$），所以 $X, Y$ 独立。

### 4. 两个随机变量函数的分布

*   **和的分布 ($Z=X+Y$)**：
    *   **卷积公式**（对于连续型，且$X, Y$独立）：
        $$ f_Z(z) = \int_{-\infty}^{+\infty} f_X(x) f_Y(z-x)dx $$
    *   **重要结论**：正态分布的可加性。若 $X \sim N(\mu_1, \sigma_1^2), Y \sim N(\mu_2, \sigma_2^2)$ 且独立，则 $X+Y \sim N(\mu_1+\mu_2, \sigma_1^2+\sigma_2^2)$。
    *   **例题3.6**：设$X, Y$独立，都服从标准正态分布$N(0,1)$。求$Z=X+Y$的概率密度。
        根据正态分布的可加性，由于$X \sim N(0,1), Y \sim N(0,1)$ 且独立，则
        $Z = X+Y \sim N(0+0, 1+1) \sim N(0,2)$。
        其概率密度函数为 $f_Z(z) = \frac{1}{\sqrt{2\pi}\sqrt{2}} e^{-\frac{z^2}{2 \times 2}} = \frac{1}{2\sqrt{\pi}} e^{-\frac{z^2}{4}}$。
*   **最大值/最小值分布**：
    设 $X, Y$ 独立，分布函数分别为 $F_X, F_Y$。
    *   $M = \max(X, Y)$ 的分布函数：$F_{max}(z) = P\{\max(X,Y) \le z\} = P\{X \le z, Y \le z\} = P\{X \le z\}P\{Y \le z\} = F_X(z)F_Y(z)$。
    *   $N = \min(X, Y)$ 的分布函数：$F_{min}(z) = P\{\min(X,Y) \le z\} = 1 - P\{\min(X,Y) > z\} = 1 - P\{X > z, Y > z\} = 1 - P\{X > z\}P\{Y > z\} = 1 - [1-F_X(z)][1-F_Y(z)]$。

## 第四章：随机变量的数字特征

这一章公式多，计算量大，是**考试计算题**的重点。

### 1. 数学期望 $E(X)$ (均值)

*   **定义**：
    *   离散型：$E(X) = \sum_k x_k p_k$。
    *   连续型：$E(X) = \int_{-\infty}^{+\infty} x f(x)dx$。
*   **随机变量函数的期望 $E[g(X)]$**：
    *   离散型：$E[g(X)] = \sum_k g(x_k) p_k$。
    *   连续型：$E[g(X)] = \int_{-\infty}^{+\infty} g(x) f(x)dx$。
    *   **例题4.1**：接例题2.7，求 $E(X)$ 和 $E(Y)$。
        $E(X) = (-1)\times0.2 + 0\times0.3 + 1\times0.4 + 2\times0.1 = -0.2 + 0 + 0.4 + 0.2 = 0.4$。
        $E(Y) = 0\times0.3 + 1\times0.6 + 4\times0.1 = 0 + 0.6 + 0.4 = 1.0$。
*   **性质**：
    *   $E(C) = C$ (C为常数)。
    *   $E(aX+b) = aE(X) + b$。
    *   **线性性质**：$E(aX+bY) = aE(X) + bE(Y)$。
    *   **重要结论**：若 $X, Y$ 独立，则 $E(XY) = E(X)E(Y)$。

### 2. 方差 $D(X)$ 或 $Var(X)$

*   **定义**：$D(X) = E[(X - E(X))^2]$。它衡量随机变量取值在其期望附近的离散程度。
*   **计算公式**（口诀：平方的期望减去期望的平方）：
    $$ D(X) = E(X^2) - [E(X)]^2 $$
    *   **例题4.2**：接例题2.7，求 $D(X)$ 和 $D(Y)$。
        $E(X)=0.4$。
        $E(X^2) = (-1)^2\times0.2 + 0^2\times0.3 + 1^2\times0.4 + 2^2\times0.1 = 0.2 + 0 + 0.4 + 0.4 = 1.0$。
        $D(X) = E(X^2) - [E(X)]^2 = 1.0 - (0.4)^2 = 1.0 - 0.16 = 0.84$。
        $E(Y)=1.0$。
        $E(Y^2) = 0^2\times0.3 + 1^2\times0.6 + 4^2\times0.1 = 0 + 0.6 + 1.6 = 2.2$。
        $D(Y) = E(Y^2) - [E(Y)]^2 = 2.2 - (1.0)^2 = 2.2 - 1.0 = 1.2$。
*   **性质**：
    *   $D(C) = 0$ (C为常数)。
    *   $D(aX+b) = a^2 D(X)$。
    *   **重要结论**：若 $X, Y$ 独立，则 $D(X \pm Y) = D(X) + D(Y)$。

### 3. 常见分布的期望与方差 (**必须熟记**)

| 分布 | 符号 | 期望 $E(X)$ | 方差 $D(X)$ |
| :--- | :--- | :--- | :--- |
| **0-1分布** | $B(1, p)$ | $p$ | $p(1-p)$ |
| **二项分布** | $B(n, p)$ | $np$ | $np(1-p)$ |
| **泊松分布** | $P(\lambda)$ | $\lambda$ | $\lambda$ |
| **均匀分布** | $U(a, b)$ | $\frac{a+b}{2}$ | $\frac{(b-a)^2}{12}$ |
| **指数分布** | $E(\lambda)$ | $\frac{1}{\lambda}$ | $\frac{1}{\lambda^2}$ |
| **正态分布** | $N(\mu, \sigma^2)$ | $\mu$ | $\sigma^2$ |

### 4. 协方差与相关系数

*   **协方差**：$Cov(X, Y) = E[(X - E(X))(Y - E(Y))]$。
    *   计算公式：$Cov(X, Y) = E(XY) - E(X)E(Y)$。
*   **相关系数**：$\rho_{XY} = \frac{Cov(X, Y)}{\sqrt{D(X)}\sqrt{D(Y)}}$。
    *   性质：
        1.  $|\rho_{XY}| \le 1$。
        2.  $|\rho_{XY}|=1$ 当且仅当 $Y$ 与 $X$ 之间存在线性关系 $Y = aX+b$。
        3.  $\rho_{XY}=0$ 称为**不相关**。
*   **关系**：
    *   **若 $X, Y$ 独立，则 $Cov(X, Y) = 0 \implies \rho_{XY} = 0$，即独立一定不相关。**
    *   **不相关不一定独立**。但对于**二维正态分布**，不相关 $\iff$ 独立。
    *   **例题4.3**：设 $(X,Y)$ 的联合分布律为：
    
        | $X \setminus Y$ | -1 | 1 |
        | :-------------- | :- | :- |
        | 0 | 0.2 | 0.3 |
        | 1 | 0.4 | 0.1 |
        求 $E(X), E(Y), D(X), D(Y), Cov(X,Y), \rho_{XY}$。
        *   $X$的边缘分布：$P\{X=0\}=0.5, P\{X=1\}=0.5$。
        *   $Y$的边缘分布：$P\{Y=-1\}=0.6, P\{Y=1\}=0.4$。
        *   $E(X) = 0\times0.5 + 1\times0.5 = 0.5$。
        *   $E(Y) = (-1)\times0.6 + 1\times0.4 = -0.6 + 0.4 = -0.2$。
        *   $E(X^2) = 0^2\times0.5 + 1^2\times0.5 = 0.5$。
        *   $D(X) = 0.5 - (0.5)^2 = 0.25$。
        *   $E(Y^2) = (-1)^2\times0.6 + 1^2\times0.4 = 0.6 + 0.4 = 1.0$。
        *   $D(Y) = 1.0 - (-0.2)^2 = 1.0 - 0.04 = 0.96$。
        *   $E(XY) = (0)\times(-1)\times0.2 + (0)\times(1)\times0.3 + (1)\times(-1)\times0.4 + (1)\times(1)\times0.1 = 0 + 0 - 0.4 + 0.1 = -0.3$。
        *   $Cov(X,Y) = E(XY) - E(X)E(Y) = -0.3 - (0.5)\times(-0.2) = -0.3 - (-0.1) = -0.2$。
        *   $\rho_{XY} = \frac{-0.2}{\sqrt{0.25}\sqrt{0.96}} = \frac{-0.2}{0.5 \times \sqrt{0.96}} = \frac{-0.4}{\sqrt{0.96}} \approx -0.408$。

## 第五章：大数定律与中心极限定理

本章主要介绍随机变量序列的渐近性质。

### 1. 大数定律 (Law of Large Numbers)

*   **切比雪夫不等式**：对于任意 $\varepsilon > 0$，有
    $$ P\{|X - E(X)| \ge \varepsilon\} \le \frac{D(X)}{\varepsilon^2} $$
    或等价形式：
    $$ P\{|X - E(X)| < \varepsilon\} \ge 1 - \frac{D(X)}{\varepsilon^2} $$
    *   **例题5.1**：某次考试，全体考生平均分是80分，标准差是5分。随机抽取一名考生，问其分数在70到90分之间的概率至少是多少？
        设$X$为考生分数，$E(X)=80, D(X)=5^2=25$。
        求 $P\{70 < X < 90\}$。
        $|X - 80| < 10$，所以 $\varepsilon=10$。
        $P\{|X-80| < 10\} \ge 1 - \frac{25}{10^2} = 1 - \frac{25}{100} = 1 - 0.25 = 0.75$。
        概率至少为0.75。
*   **辛钦大数定律**：
    设 $X_1, X_2, ..., X_n$ 是独立同分布的随机变量序列，且 $E(X_k)=\mu$ 存在。则当 $n \to \infty$ 时，其样本均值 $\bar{X}_n = \frac{1}{n}\sum_{k=1}^n X_k$ 依概率收敛于数学期望 $\mu$。
    $$ \lim_{n \to \infty} P\{|\bar{X}_n - \mu| < \varepsilon\} = 1 $$

### 2. 中心极限定理 (CLT)

描述和的分布的稳定性。
*   **独立同分布的CLT** (Lindeberg-Levy)：
    设 $X_1, X_2, ..., X_n$ 是独立同分布的随机变量序列，且 $E(X_k)=\mu, D(X_k)=\sigma^2$ 都存在。则当 $n$ 充分大时，其标准化后的和（或均值）近似服从标准正态分布 $N(0,1)$。
    $$ Y_n = \frac{\sum_{k=1}^n X_k - n\mu}{\sqrt{n}\sigma} \approx N(0,1) $$
    或
    $$ Z_n = \frac{\bar{X}_n - \mu}{\sigma/\sqrt{n}} \approx N(0,1) $$
    *   **例题5.2**：某机器生产的零件长度平均为50mm，标准差为2mm。现在随机抽取100个零件，求这100个零件的平均长度在49.8mm到50.2mm之间的概率。
        设零件长度 $X_i$，总体均值 $\mu=50$，总体标准差 $\sigma=2$。
        样本量 $n=100$。样本均值 $\bar{X}$。
        根据CLT，$\bar{X} \approx N(\mu, \sigma^2/n) = N(50, 2^2/100) = N(50, 0.04)$。
        标准化：$Z = \frac{\bar{X}-50}{\sqrt{0.04}} = \frac{\bar{X}-50}{0.2} \sim N(0,1)$。
        $P\{49.8 < \bar{X} < 50.2\} = P\{\frac{49.8-50}{0.2} < Z < \frac{50.2-50}{0.2}\}$
        $= P\{-1 < Z < 1\} = \Phi(1) - \Phi(-1) = 2\Phi(1) - 1$
        $= 2 \times 0.8413 - 1 = 0.6826$。
*   **德莫佛-拉普拉斯定理**：
    二项分布 $B(n, p)$ 当 $n$ 充分大时，近似于正态分布 $N(np, np(1-p))$。
    $$ \frac{X - np}{\sqrt{np(1-p)}} \approx N(0,1) $$
    *   **例题5.3**：掷一枚均匀硬币100次，求正面出现次数在45到55次之间的概率。
        设$X$为正面出现的次数，$X \sim B(100, 0.5)$。
        $n=100, p=0.5$。
        $E(X) = np = 100 \times 0.5 = 50$。
        $D(X) = np(1-p) = 100 \times 0.5 \times 0.5 = 25$，$\sqrt{D(X)} = 5$。
        标准化：$Z = \frac{X-50}{5} \sim N(0,1)$。
        $P\{45 \le X \le 55\} = P\{\frac{45-50}{5} \le Z \le \frac{55-50}{5}\}$
        $= P\{-1 \le Z \le 1\} = \Phi(1) - \Phi(-1) = 2\Phi(1) - 1 = 0.6826$。
        （*注：实际使用时通常会加入0.5的连续性修正，即 $P\{44.5 \le X \le 55.5\}$，但这里为了与课件计算保持一致，暂不使用。*）

## 第六章：数理统计基础与参数估计

这一章是统计推断的核心，掌握**估计方法**和**区间估计的公式**。

### 1. 总体与样本

*   **总体**：研究对象的全体。通常用其分布$F(x;\theta)$来描述，$\theta$为未知参数。
*   **个体**：总体中的每一个元素。
*   **样本**：从总体中抽取的一部分个体，$X_1, X_2, ..., X_n$。
    *   **随机样本**：独立同分布（i.i.d）的样本。

### 2. 统计量

*   **定义**：样本的函数，且不依赖于任何未知参数。
    *   **例题6.1**：设 $X_1, X_2, X_3$ 是来自 $N(\mu, \sigma^2)$ 的样本。判断下列哪些是统计量。
        1.  $T_1 = X_1+X_2e^{X_3}$：是统计量（只依赖样本）。
        2.  $T_2 = \bar{X} - 2\mu$：不是统计量（包含未知参数 $\mu$）。
        3.  $T_3 = S^2/\sigma^2$：不是统计量（包含未知参数 $\sigma^2$）。
        4.  $T_4 = X_1^2 + X_2^2 + X_3^2$：是统计量。
*   **常用统计量**：
    *   **样本均值**：$\bar{X} = \frac{1}{n}\sum X_i$。
    *   **样本方差**：$S^2 = \frac{1}{n-1}\sum (X_i - \bar{X})^2$。（**注意分母是$n-1$，是无偏估计量**）

### 3. 三大抽样分布

*   **$\chi^2$ 分布 (卡方分布)**：$Y = \sum_{i=1}^n Z_i^2 \sim \chi^2(n)$，其中 $Z_i \sim N(0,1)$ 且相互独立，$n$ 为自由度。
    *   性质：$E(Y)=n, D(Y)=2n$。
*   **$t$ 分布 (学生t分布)**：$T = \frac{Z}{\sqrt{Y/n}} \sim t(n)$，其中 $Z \sim N(0,1)$, $Y \sim \chi^2(n)$ 且 $Z, Y$ 独立，$n$ 为自由度。
*   **$F$ 分布**：$F = \frac{Y_1/n_1}{Y_2/n_2} \sim F(n_1, n_2)$，其中 $Y_1 \sim \chi^2(n_1)$, $Y_2 \sim \chi^2(n_2)$ 且 $Y_1, Y_2$ 独立，$n_1, n_2$ 为自由度。

### 4. 正态总体的抽样分布性质 (**非常重要，用于构建区间估计**)

设 $X_1, ..., X_n$ 是来自正态总体 $N(\mu, \sigma^2)$ 的随机样本，$\bar{X}$ 是样本均值，$S^2$ 是样本方差。
1.  **$\bar{X}$ 的分布**：$\bar{X} \sim N(\mu, \sigma^2/n)$。
2.  **$S^2$ 的分布**：$\frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$。
3.  **独立性**：$\bar{X}$ 与 $S^2$ 相互独立。
4.  **$t$ 统计量**：$\frac{\bar{X} - \mu}{S/\sqrt{n}} \sim t(n-1)$。

### 5. 参数的点估计

*   **矩估计法**：
    *   原理：用样本矩替换总体矩来估计未知参数。
    *   步骤：
        1.  计算总体矩 $E(X^k)$（包含未知参数）。
        2.  计算样本矩 $\frac{1}{n}\sum X_i^k$。
        3.  令样本矩等于总体矩，解方程组得到参数的估计值。
    *   **例题6.2**：设 $X_1, ..., X_n$ 是来自 $U(0, \theta)$ 的样本，$\theta>0$ 未知。用矩估计法估计 $\theta$。
        $E(X) = \frac{0+\theta}{2} = \frac{\theta}{2}$。
        样本一阶矩 $\bar{X} = \frac{1}{n}\sum X_i$。
        令 $\bar{X} = \frac{\theta}{2}$，解得 $\hat{\theta} = 2\bar{X}$。
*   **最大似然估计法 (MLE)**：
    *   原理：选取参数值，使得当前样本出现的概率（似然函数）最大。
    *   步骤：
        1.  写出似然函数 $L(\theta)$：
            *   离散：$L(\theta) = P(X_1=x_1,...,X_n=x_n;\theta) = \prod_{i=1}^n P(X_i=x_i;\theta)$。
            *   连续：$L(\theta) = f(x_1,...,x_n;\theta) = \prod_{i=1}^n f(x_i;\theta)$。
        2.  取对数似然函数 $\ln L(\theta)$。
        3.  对 $\theta$ 求偏导数并令其为0：$\frac{d}{d\theta} \ln L(\theta) = 0$。
        4.  解方程得到参数的估计值 $\hat{\theta}$。
    *   **例题6.3**：设 $X_1, ..., X_n$ 是来自 $B(1, p)$ 的样本，$p$ 未知。用MLE估计 $p$。
        $P\{X=x\} = p^x (1-p)^{1-x}$ ($x=0,1$)。
        $L(p) = \prod_{i=1}^n p^{x_i} (1-p)^{1-x_i} = p^{\sum x_i} (1-p)^{n-\sum x_i}$。
        $\ln L(p) = (\sum x_i) \ln p + (n-\sum x_i) \ln (1-p)$。
        $\frac{d}{dp} \ln L(p) = \frac{\sum x_i}{p} - \frac{n-\sum x_i}{1-p} = 0$。
        解得 $\hat{p} = \frac{\sum x_i}{n} = \bar{X}$。

### 6. 估计量的评价标准

*   **无偏性**：若 $E(\hat{\theta}) = \theta$，则 $\hat{\theta}$ 是 $\theta$ 的无偏估计量。
    *   **例题6.4**：样本均值 $\bar{X}$ 是总体均值 $\mu$ 的无偏估计量：$E(\bar{X}) = E(\frac{1}{n}\sum X_i) = \frac{1}{n}\sum E(X_i) = \frac{1}{n}\sum \mu = \mu$。
    *   **例题6.5**：样本方差 $S^2 = \frac{1}{n-1}\sum (X_i - \bar{X})^2$ 是总体方差 $\sigma^2$ 的无偏估计量：$E(S^2) = \sigma^2$。
*   **有效性**：在所有无偏估计量中，方差最小的估计量是有效估计量。方差越小，估计精度越高。
*   **相合性 (一致性)**：若当样本量 $n \to \infty$ 时，估计量 $\hat{\theta}$ 依概率收敛于参数真值 $\theta$，则 $\hat{\theta}$ 是 $\theta$ 的相合估计量。

### 7. 参数的区间估计

构建**置信区间** $(\underline{\theta}, \bar{\theta})$，使得 $P\{ \underline{\theta} < \theta < \bar{\theta} \} = 1-\alpha$，$1-\alpha$ 称为**置信水平**。

*   **总体均值 $\mu$ 的区间估计**：
    1.  **总体方差 $\sigma^2$ 已知**：
        *   枢轴量：$Z = \frac{\bar{X}-\mu}{\sigma/\sqrt{n}} \sim N(0,1)$。
        *   置信区间：$\left( \bar{X} - Z_{\alpha/2} \frac{\sigma}{\sqrt{n}}, \bar{X} + Z_{\alpha/2} \frac{\sigma}{\sqrt{n}} \right)$。
        *   **例题6.6**：某批糖果重量 $\mu$ 未知，$\sigma=10$ 克。抽取12包糖果，样本均值 $\bar{x}=502.92$ 克。求 $\mu$ 的 95% 置信区间。
            $\alpha=0.05 \implies \alpha/2 = 0.025$。查表 $Z_{0.025}=1.96$。
            置信区间为 $\left( 502.92 - 1.96 \frac{10}{\sqrt{12}}, 502.92 + 1.96 \frac{10}{\sqrt{12}} \right)$。
            计算得 $\left( 502.92 - 5.658, 502.92 + 5.658 \right) = \left( 497.26, 508.58 \right)$。
    2.  **总体方差 $\sigma^2$ 未知**：
        *   枢轴量：$T = \frac{\bar{X}-\mu}{S/\sqrt{n}} \sim t(n-1)$。
        *   置信区间：$\left( \bar{X} - t_{\alpha/2}(n-1) \frac{S}{\sqrt{n}}, \bar{X} + t_{\alpha/2}(n-1) \frac{S}{\sqrt{n}} \right)$。
        *   **例题6.7**：接上例，若 $\sigma$ 未知，样本方差 $S=6.2022$。求 $\mu$ 的 95% 置信区间。
            $n=12, n-1=11$。 $\alpha=0.05 \implies \alpha/2 = 0.025$。查表 $t_{0.025}(11)=2.201$。
            置信区间为 $\left( 502.92 - 2.201 \frac{6.2022}{\sqrt{12}}, 502.92 + 2.201 \frac{6.2022}{\sqrt{12}} \right)$。
            计算得 $\left( 502.92 - 3.94, 502.92 + 3.94 \right) = \left( 498.98, 506.86 \right)$。

*   **总体方差 $\sigma^2$ 的区间估计**：
    1.  **总体均值 $\mu$ 未知**：
        *   枢轴量：$\chi^2 = \frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$。
        *   置信区间：$\left( \frac{(n-1)S^2}{\chi^2_{\alpha/2}(n-1)}, \frac{(n-1)S^2}{\chi^2_{1-\alpha/2}(n-1)} \right)$。
        *   **例题6.8**：接例题6.7，求 $\sigma^2$ 的 95% 置信区间。
            $n=12, n-1=11$。 $\alpha=0.05 \implies \alpha/2 = 0.025, 1-\alpha/2=0.975$。
            查表 $\chi^2_{0.025}(11)=21.920, \chi^2_{0.975}(11)=3.816$。
            置信区间为 $\left( \frac{11 \times (6.2022)^2}{21.920}, \frac{11 \times (6.2022)^2}{3.816} \right)$。
            计算得 $\left( \frac{11 \times 38.467}{21.920}, \frac{11 \times 38.467}{3.816} \right) = \left( 19.33, 110.87 \right)$。
